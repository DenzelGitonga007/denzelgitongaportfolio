# Denzel Gitonga Portfolio

Modern, high-performance portfolio built with Next.js, Framer Motion, and Tailwind CSS.

## Features
- **Dynamic Hero Section**: Interactive typewriter effect and premium background animations.
- **Direct Connect**: Strategic communication grid for high-priority outreach.
- **Responsive Mastery**: Fully optimized for all device sizes.
- **Static Export**: Optimized for GitHub Pages hosting.

## Deployment
This project is configured for GitHub Pages using the `gh-pages` package.

To deploy the latest version:
```bash
npm run deploy
```

## Structure
- `/src`: Core application logic and premium components.
- `/public`: Branding assets and icons.
- `legacy-bootstrap`: Archived in the `legacy-bootstrap-archive` branch.
